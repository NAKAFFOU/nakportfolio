#YASAI Simulation

    YASAI is a simulation add-on for Microsoft Excel, designed for teaching Monte Carlo simulation to beginning students. It is distinguished by its ease of use, user-friendly interface, and ability to generate random variables from various probability distributions using spreadsheet functions. YASAI allows users to specify model outputs using the SIMOUTPUT function and offers flexible scenario management with the PARAMETER function, allowing for testing different parameter combinations. While not as feature-rich as some commercial packages such as @Risk or Crystal Ball, YASAI's simplicity, accessibility, and formula-driven design make it a valuable tool for teaching basic simulation concepts.
